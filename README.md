# 💱 Currency Converter

A simple and responsive Currency Converter web application that allows users to convert currencies in real-time using exchange rate APIs. The project provides an intuitive interface for selecting currencies, entering an amount, and instantly viewing the converted value.

## 🚀 Features

- Convert between multiple international currencies
- Real-time exchange rates
- Country flags displayed for selected currencies
- User-friendly and responsive UI
- Swap currencies easily
- Input validation for better user experience
- Fast and lightweight application

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Currency Exchange API
- Flags API

## ⚙️ How It Works

1. Enter the amount to convert.
2. Select the source currency.
3. Select the target currency.
4. Click the **Get Exchange Rate** button.
5. The converted amount is displayed instantly.

## 🔧 Installation

### Clone the Repository

```bash
git clone https://github.com/spandanreddymutyala/Currency-Converter.git
```

### Navigate to the Project Directory

```bash
cd Currency-Converter
```

### Run the Project

Simply open `index.html` in your browser.

## 🌐 API Used

Exchange rates are fetched using:

- Fawaz Ahmed Currency API
- Country Flags API

Example API Endpoint:

```javascript
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies
```
